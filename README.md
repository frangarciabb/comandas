# Bot Bar El Mostacho

## Archivos
- `bot.py` — lógica completa del bot
- `requirements.txt` — dependencias Python
- `Procfile` — instrucciones para Render
- `.env.example` — variables de entorno a completar

## Deploy en Render
1. Subí esta carpeta a GitHub
2. Creá un Web Service en render.com apuntando al repo
3. Completá las variables de entorno en el panel de Render
4. Copiá la URL del servicio y pegala en Twilio como webhook

## Webhook URL
https://TU-APP.onrender.com/webhook
